## サービス概要

「リライトログ」は、「特定のキーワードの検索順位の変動を監視するのが面倒」という問題を解決したい、ブログ運営者向けの「検索順位チェックツール」です。

ユーザーは、「特定のキーワードの検索順位と記事URLを記録し、変更点についてのコメントを残すことができます。

## Setup

```
$ bin/setup
$ rails server
```

テストデータが生成されない場合。

```
$ rails db:seed
```

ローカルで検索順位等のデータをみたいとき。

- email: testtest@example.com
- pass: testtest

## テスト
ヘッドレスブラウザーでテスト
```
$ ./bin/test
```

## Lint

```
$ ./bin/lint
```
